## Rejection Reason

The current kept helper is a replay stage: it needs reflectors already generated by `deterministic_symmetric_full_to_band_reflectors`. That generator is still scalar and produces the reference reduced matrix while collecting reflectors. A direct public route built from the current helpers would either:

1. run scalar full-to-band generation and compact replay, duplicating the reduction work, or
2. call `eig_banded`, which expands banded storage back to a dense matrix and performs another scalar tridiagonalization before solving.

Neither is the intended public algorithm, and neither satisfies the one-lever public dispatch contract. The missing primitive is not a small wiring edit; it is fused compact-WY reflector generation plus a real band-to-tridiagonal solve/backtransform path.
